# google-music-play-around
UI wrapper for Google Music
based on https://github.com/twostairs/loop

### Install
```
npm install
```
### run
```
npm run start
```
### Build
Requirement `npm install electron-packager -g`
```
electron-packager . play --platform=darwin --arch=x64 --overwrite=true
```
